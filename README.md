<h1>Extract text from PDF</h1>
<hr>
This simple app show text in a window extracted from .pdf file.
<hr>

<h3>Modules used →</h3>
▶ PyPDF2<br>
▶ Tkinter
<hr/>
<h3>How to run →</h3>
● Install module named 'PyPDF2'<br><br>
<pre>pip install pypdf2</pre><br>
● run the file pdfToText.py<hr>
